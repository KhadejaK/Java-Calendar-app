# Java-Calendar-app

A Calendar app that allows you to <br />
<ul>
  <li>View calendar in day or month view</li>
  <li>Create a new event (one time ocurrance or regularly occuring)</li>
  <li>Go to a specific event</li>
  <li>Delete events</li>
  <li>Quit</li>
</ul>
